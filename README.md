##Chatbot

An AI-powered chatbot that reads documents and answers questions from them — with voice input support and fully local LLM inference. No API keys. No cloud dependency.
Built during Generative AI Internship at Infosys Springboard · Oct – Dec 2025

##Overview
Upload an image or scanned document, ask a question by voice or text, and get an answer grounded in the content of your document — not general knowledge.
The pipeline: image upload → OCR text extraction → natural language question → local LLM response.

##Features

Document OCR — extracts text from uploaded images using Tesseract and OpenCV preprocessing
Voice input — ask questions by speaking using the built-in SpeechRecognition integration
Local LLM — answers generated by Ollama running entirely on your machine
Chat history — conversation persisted across sessions in chat_history.json
Browser UI — clean Streamlit interface, no frontend setup required

##Prerequisites
Python 3.8+
Ollama installed and running
Tesseract OCR installed on your system

1. Clone the repository

bashgit clone https://github.com/yourusername/document-qa-chatbot.git
cd document-qa-chatbot

2. Install Python dependencies

bashpip install -r requirements.txt

3. Install Tesseract

Ubuntu / Debian
bashsudo apt install tesseract-ocr
macOS
bashbrew install tesseract
Windows
Download from UB-Mannheim/tesseract and add the install path to your system environment variables.

4. Pull an Ollama model
bashollama pull mistral

5. Run the app
bashstreamlit run chatai.py
Open http://localhost:8501 in your browser.
